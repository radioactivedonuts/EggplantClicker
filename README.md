# 🍆 EggplantClicker

A simple HTML-only clicker game where you click the eggplant to earn Eggplants and buy upgrades.

## Features

* 🍆 Click the eggplant to earn Eggplants
* ⚡ Buy upgrades to increase your earnings
* 🤖 Auto-clickers that earn Eggplants automatically
* 💾 Saves your progress in your browser
* 🎮 No downloads required
* 📄 Single-file HTML game
* 🚀 Easy to share with friends

## How to Play

1. Download the HTML file.
2. Open the file in a web browser.
3. Click the eggplant to earn Eggplants.
4. Purchase upgrades to earn Eggplants faster.
5. Try to reach huge numbers and unlock everything.

## Installation

This game consists of a single HTML file.

1. Copy the code into a text file.
2. Save the file with a `.html` extension (for example: `EggplantClicker.html`).
3. In the Save As dialog, set **Save as type** to **All Files**.
4. Open the file in your browser.

## Saving

Your progress is automatically saved using your browser's local storage.

## Sharing

You can send the HTML file to other people and they can play it in their browser.
